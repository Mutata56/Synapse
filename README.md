# Synapse

> Локальное приложение для заметок: редактор в стиле Notion и граф связей в стиле Obsidian — в одном окне.

Заметки лежат на диске обычными Markdown-файлами. Внутри — блочный редактор, [[вики-ссылки]] с обратными ссылками, теги, ежедневные заметки, календарь с синхронизацией и интерактивный граф, который показывает, как всё связано между собой. Десктоп, работает офлайн, данные остаются у вас.

**Стек:** Tauri 2 (Rust) · React · TypeScript · Vite · Zustand · BlockNote · Pixi.js

## Возможности

**Редактор**
- Блочный редактор на BlockNote с кастомными блоками: выноски (callouts), таблицы, галереи и карусели, карточки файлов, встроенные базы данных, мультиколонки.
- Интерактивная доска (whiteboard) прямо внутри заметки — фигуры, связи, свободное рисование.
- Обложки заметок, эмодзи и отметка настроения.

**Связи**
- [[Вики-ссылки]] с автодополнением, обратные ссылки (backlinks) и блок «неупомянутые связи» (unlinked mentions).
- Теги `#тег` и отдельная страница тегов.
- Интерактивный граф связей (force-directed, Pixi.js) и локальный граф вокруг открытой заметки.

**Организация**
- Дерево файлов и папок, избранное, входящие (inbox), галерея заметок.
- Командная палитра (Ctrl/Cmd + K) и быстрый поиск.
- Заметка дня и быстрая заметка по глобальному хоткею — работает, даже когда приложение свёрнуто в трей.

**Календарь**
- События и задачи, повторяющиеся задачи, цвета и метки.
- Подписка на внешний календарь по ссылке iCalendar (.ics, только чтение).
- Пуш своих задач в Яндекс.Календарь через CalDAV.

**Данные и оформление**
- Хранение в обычных `.md` — никаких баз и облаков, всё на диске.
- Корзина с превью и восстановлением, история версий, бэкап и восстановление в .zip.
- Статистика письма на отдельном дашборде.
- Тёмная тема, выбор акцентного цвета, интерфейс на русском и английском.

Готовые установщики (Linux · Windows · macOS) — на странице [Releases](https://github.com/Mutata56/notes-app/releases). Либо собрать самому, см. ниже.

## Разработка (Dev)

```bash
npm install
npm run tauri dev
```

Первая сборка на Rust скачивает и компилирует все зависимости (crates) Tauri, поэтому занимает несколько минут; последующие сборки инкрементальные (быстрые).

## Сборка (Build)

```bash
npm run tauri build
```

Установщики окажутся в `src-tauri/target/release/bundle/`. На каждую ОС собираются свои форматы (`.AppImage`/`.deb`/`.rpm`, `.msi`/`.exe`, `.dmg`).

## Где хранятся заметки

Обычные `.md` файлы в директории данных приложения:

- Linux: `~/.local/share/com.kirill.synapse/notes`
- Windows: `%APPDATA%\com.kirill.synapse\notes`
- macOS: `~/Library/Application Support/com.kirill.synapse/notes`
